üìÅ How to create a config file for your MCP server

Create a JSON file (e.g., my-server.json) with this format:

{
  "mcpServers": {
    "my-server": {
      "command": "python",
      "args": ["-m", "my_mcp_server"]
    }
  }
}

Or for an SSE server already running:

{
  "mcpServers": {
    "my-server": {
      "url": "http://localhost:3000/sse"
    }
  }
}

You can also use your existing Claude Desktop config:

mcp-scan scan --config ~/.config/Claude/claude_desktop_config.json --json
